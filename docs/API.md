## Group Assets

### Image [POST /api/v1/images]

- バイナリデータを直接送信します
- アップロード上限は 10 MB です

- Request (application/octet-stream)

- Response 200 (application/json)

  ```json
  {
    "id": "538dbca6-85d6-434e-a1f4-b370d03dbb85"
  }
  ```
